# Stock Predict Skill

本 skill 调用后端 API 分析股票，基于"买入预期，卖出事实"原则给出投资建议。

## 分析维度

1. **技术因子** (20%): 均线趋势、MACD、KDJ、RSI、布林带、形态识别
2. **宏观因子** (20%): 美联储政策、汇率、CPI/PMI、地缘政治、VIX
3. **资金面因子** (15%): 北向资金、融资融券、主力净流入
4. **行业因子** (15%): 行业趋势、上游价格、行业政策
5. **基本面因子** (15%): 业绩预告、财报日期、估值分位
6. **周期因子** (10%): 美林时钟、季节性效应
7. **事件驱动** (5%): 财报日期、政策事件、分红

## 工作流程

1. 解析股票代码（自动识别 A股/港股/美股）
2. 调用后端 API 获取行情和数据
3. AI 综合分析（MiniMax 模型）
4. 返回拐点预测和建议
